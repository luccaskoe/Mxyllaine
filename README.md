# 💎 MAXYLLAINE - Site de Estética de Luxo

Website moderno e responsivo para clínica de estética com integração de IA via API Groq.

## 🎯 Recursos Principais

- ✨ **Design Minimalista e Sofisticado** - Paleta de cores: Nude, Dourado Fosco e Branco
- 🤖 **Chatbot IA Inteligente** - Assistente virtual 24/7 com API Groq
- 📱 **Totalmente Responsivo** - Otimizado para todos os tamanhos de tela
- 🎨 **Interface Moderna** - Animações suaves e transições elegantes
- 🏥 **Galeria Antes/Depois** - Showcases interativos dos resultados
- 📋 **FAQ Dinâmico** - Acordeão interativo com respostas detalhadas
- 📞 **Múltiplos Canais de Contato** - WhatsApp, Email, Telefone

## 📋 Procedimentos Oferecidos

1. **Botox** - Suavização de rugas e prevenção de expressão
2. **Lipo de Papada** - Redução de gordura localizada
3. **Micropigmentação** - Design semi-permanente de sobrancelhas
4. **Design de Sobrancelhas** - Arquitetura facial personalizada
